# Daangn

당근마켓 클론 코딩 서버

---

## 프로젝트 소개

중고거래 플랫폼 **당근마켓의 클론 코딩 서버**입니다.

대규모 트래픽이 발생하는 콘텐츠 플랫폼을 안정적으로 운영하기 위한
설계·구현·운영 경험을 목표로 진행한 백엔드 프로젝트입니다.

### 📅 진행 기간

| 단계      | 기간                           | 내용         |
|---------|------------------------------|------------|
| 1차 (기본) | 2026.04.17 ~ 2026.05.20 (5주) | MVP 구현     |
| 2차 (심화) | 2026.05.21 ~                 | 고도화, 성능 개선 |

### 👥 팀 구성

- 백엔드 3명 (강상욱, 김재성, 차민혁)

### 🛠 주요 기능

- 회원 — JWT 기반 인증/인가, Spring Security
- 상품 거래 — 게시글 CRUD
- 채팅 — WebSocket STOMP 기반 1:1, 멀티 채팅
- 알림 — SSE 구독

| Infra     | Docker, AWS       |

## 로컬 실행 방법

### 1. 인프라 실행

```bash
docker compose up -d
```

### 2. 애플리케이션 실행

IDE에서 `SPRING_PROFILES_ACTIVE=dev`로 실행하거나:

```bash
SPRING_PROFILES_ACTIVE=dev ./gradlew bootRun
```

| 엔드포인트    | 주소                                      |
|----------|-----------------------------------------|
| API      | `http://localhost:8080`                 |
| API Docs | `http://localhost:8080/docs/index.html` |
